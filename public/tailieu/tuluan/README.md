# ✍️ Tài liệu Tự luận

Đặt các file bài tập tự luận kèm đáp án vào thư mục này.

## Định dạng hỗ trợ
- `.pdf`, `.docx`, `.txt`, `.md`

## Nội dung gợi ý
- Các dạng bài tập tự luận thường gặp
- Bài tập tính toán hóa học
- Bài tập viết phương trình phản ứng
- Bài tập giải thích hiện tượng
- Đáp án và hướng dẫn giải chi tiết
