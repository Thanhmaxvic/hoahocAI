# 📚 Tài liệu Lý thuyết

Đặt các file tài liệu lý thuyết vào thư mục này.

## Định dạng hỗ trợ
- `.pdf` — File PDF
- `.docx` — File Word
- `.txt` — File văn bản
- `.md` — File Markdown

## Nội dung gợi ý
- Lý thuyết tổng hợp chương Đại cương về Kim loại
- Tóm tắt công thức, phương trình phản ứng
- Bảng tuần hoàn các nguyên tố
- Dãy hoạt động hóa học của kim loại
