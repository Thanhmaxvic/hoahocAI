# 📝 Tài liệu Trắc nghiệm

Đặt các file câu hỏi trắc nghiệm kèm đáp án vào thư mục này.

## Định dạng hỗ trợ
- `.pdf`, `.docx`, `.txt`, `.md`

## Nội dung gợi ý
- Ngân hàng câu hỏi trắc nghiệm nhiều lựa chọn (A, B, C, D)
- Câu hỏi trắc nghiệm đúng/sai
- Câu hỏi trắc nghiệm trả lời ngắn
- Đáp án và giải thích chi tiết
